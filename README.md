# Intelligent-image-annotation-web-tool-based-on-multimodal-LLM
**Local MLLM Annotation Tool** – Offline multimodal LLM for fast pose recognition (DWpose), one-click labeling &amp; smart review. All local inference ensures data security. Features: batch processing, auto-scoring, web UI. Ideal for security, sports &amp; HCI projects needing efficient, sovereign image annotation.
